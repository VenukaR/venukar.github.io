Portfolio Website

My personal portfolio website designed to showcase my projects, skills, and journey as a Computer Science undergraduate. Features a modern, colorful design with smooth animations and responsive layout.

Features

  • Modern Design: Beautiful gradient backgrounds and smooth animations
  • Responsive Layout: Works seamlessly across all devices
  • Project Showcase: Detailed project pages with descriptions and links
  • Contact Section: Easy ways to connect via social media and email
  • Fast Performance: Built with Next.js for optimal speed

Technologies Used

  • Next.js 13 with App Router
  • Tailwind CSS for styling
  • TypeScript for type safety
  • MDX for content management
  • Framer Motion for animations
