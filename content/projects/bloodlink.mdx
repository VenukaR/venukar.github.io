## BloodLink (Group Project â€“ Idealize '23)

A web platform connecting blood donors, blood banks, and recipients to streamline the donation process and improve donation rates.

### Key Features

- **Donor Registration**: Easy sign-up for potential blood donors
- **Blood Bank Management**: Inventory tracking and management system
- **Request Matching**: Connect recipients with compatible donors
- **Notification System**: Alert donors when their blood type is needed
- **Location-Based Search**: Find nearby donors or blood banks
- **Donation History**: Track donation records and eligibility

### Technology Stack

- Modern web framework for responsive design
- Database management for donor records
- Real-time notification system
- Geolocation services
- Secure authentication

### Impact

Addresses critical blood shortage issues by creating an efficient network that reduces response time in emergencies and encourages regular donations.
