## About

This portfolio showcases my journey as a Computer Science student, featuring:
- **Vibrant Design**: Colorful gradient backgrounds using purple, cyan, and blue tones
- **Project Showcase**: Detailed project pages including Rumble Guard and more
- **Responsive Layout**: Mobile-friendly design that works on all devices
- **Social Integration**: Easy access to GitHub, LinkedIn, Twitter, and other platforms

## Tech Stack

- **Framework**: Next.js 13 with App Router
- **Styling**: Tailwind CSS with custom gradients
- **Content**: MDX for project documentation
- **Analytics**: Upstash Redis for pageview tracking
- **Animations**: Framer Motion for smooth transitions

## Running Locally

```sh-session
git clone https://github.com/venukaranasinghe/portfolio.git
cd portfolio
```

Create a `.env` file similar to `.env.example`.

Then install dependencies and run the development server:
```sh-session
npm install
npm run dev
```
